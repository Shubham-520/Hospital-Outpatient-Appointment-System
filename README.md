🏥 Hospital Outpatient Appointment System
End-to-End BA & Data Analytics Project

📌 Executive Summary
This project defines the transition from a manual, spreadsheet-based appointment process to a digital management system. The goal was to reduce patient wait times from 2 hours to under 30 minutes and lower no-show rates.

📑 Project Components
• Business Requirements (BRD): Detailed functional and non-functional requirements.

• Visual Modeling: Process flows (As-Is/To-Be) and Use Case diagrams to map the user journey.

• Technical Analysis: SQL queries for data extraction and metric calculation.

• Data Visualization: Interactive Power BI dashboard tracking KPIs like Doctor Utilization and Monthly Trends.

🔑 Key Insights from Dashboard
• Wait Time Reduction: Formula established to track average minutes per patient.

• No-Show Tracking: Identified Cardiology and Pediatrics as high-risk departments for skips.

• Utilization: Monitored doctor workload to ensure efficient scheduling.
